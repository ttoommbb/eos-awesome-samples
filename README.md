# Getting started

For compile and deploy contracts you must see instructions inside [Ping](docs/Ping/README.md) and [ToDo](docs/ToDo/README.md). Then:

```
cd frontend
cp -a env.template .env.local
yarn install
yarn start
```

Finally, open http://localhost:3000 at browser
